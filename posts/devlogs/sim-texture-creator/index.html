Sim Texture Creator

One of the tasks I had to do during Visual Design courses was to create a portrait of something, without any constraints. I thought that realizing a portrait using my programming skill would be very fun and interesting and I decided to do so. Wishing to learn more about texture generation methods, I created this Sim Texture Creator. This “game” creates a “portrait” texture relative to the name and character traits of a person. I reused the character traits that we can find in the game The Sims 2. Obviously, it’s *my* portrait of a person. Everything is based on my personal feeling. For example, for a serious person, the algorithm will use value noise, because I found that it produced more checkered textures. A shy person will have a low-resolution texture, etc…
